# Contributing

## Development

- Use Node.js 18 or later.
- Install dependencies with `npm install`.
- Run `npm run build`, `npm run lint`, and `npm test` before opening a change.

## Pull Requests

- Keep changes focused and include tests for user-visible behavior.
- Update documentation when command behavior or configuration changes.
- Describe any Salesforce-specific assumptions in the pull request.

## Reporting Issues

- Include the command used, the config file, and a minimal project layout that reproduces the issue.
- Redact org names, credentials, or proprietary metadata before sharing logs.
